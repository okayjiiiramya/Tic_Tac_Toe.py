# 🎮 Tic Tac Toe Game in Python

A simple console-based Tic Tac Toe game developed using Python. This project allows two players to play Tic Tac Toe in the terminal while demonstrating Python fundamentals such as loops, functions, conditionals, and game logic.

## 🚀 Features

- Two-player gameplay (X and O)
- Interactive console interface
- Winner detection
- Draw detection
- Input validation
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- Git
- GitHub

🎯 Game Rules
Player 1 uses X
Player 2 uses O
Players take turns choosing positions on the board.
The first player to align three symbols horizontally, vertically, or diagonally wins.
If all positions are filled and no player wins, the game ends in a draw.
